# PrototypeWorkQueueView

Work queue view that renders the ItemList component with all its built-in grouping, filtering, toolbar actions, and display modes.

## Import

```tsx
import { PrototypeWorkQueueView } from "@handled-ai/design-system"
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `headerActions` | `ReactNode` | Extra content rendered at the top-right of the view. |

## Dependencies

- `ItemList`
